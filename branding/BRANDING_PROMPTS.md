# ChangeLog - Branding Prompts

## 1. TITLE CARD (16:9 Landscape - 1920x1080px)

**For:** GitHub repository header image, presentations, documentation

**DALL-E Prompt:**
```
Create a sleek, modern title card for "ChangeLog" - an automated changelog generator tool.

STYLE:
- Dark navy blue background (#0a0e27)
- Bright cyan accents (#00d4ff)
- Tech-forward, professional aesthetic
- Minimalist geometric patterns

LAYOUT:
- Large bold title "CHANGELOG" centered
- Smaller tagline below: "Transform Git History Into Professional Changelogs"
- Visual element: A cascading timeline of git commits flowing into an organized document
- Subtle circuit board patterns in background with git branch-like connections
- Small version tags (v1.0, v2.0) floating along the timeline

MOOD: Professional, cutting-edge, reliable, organized, Team Brain ecosystem
```
